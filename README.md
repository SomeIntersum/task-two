# Инструкция по установке и настройке Django-проекта

1. **Установка зависимостей**

   Установите все зависимости, указанные в `requirements.txt`:

    ```sh
    pip install -r requirements.txt
    ```

2. **Создание миграций**

    Создайте файлы миграций на основе изменений в моделях:

    ```sh
    python manage.py makemigrations
    ```

3. **Применение миграций**

   Примените миграции для создания или обновления структуры базы данных:

    ```sh
    python manage.py migrate
    ```

4. **Создание суперпользователя**

   Создайте суперпользователя для доступа к административной панели Django:

    ```sh
    python manage.py createsuperuser
    ```

   Следуйте инструкциям на экране для ввода имени пользователя, адреса электронной почты и пароля.

5. **Запуск сервера разработки**

   Запустите сервер разработки:

    ```sh
    python manage.py runserver
    ```

6. **Открытие вашего сайта**

   Откройте следующие URL в вашем браузере:

    - Административная панель: [http://127.0.0.1:8000/admin](http://127.0.0.1:8000/admin)
    - Главная страница: [http://127.0.0.1:8000](http://127.0.0.1:8000)
